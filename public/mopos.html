Charging (over HTTP) API

Operation URL Method Params details Response Response codes Additional information
Clear funds http://apps.lsim.az/mopos/v1/charge?
login=LOGIN
&msisdn=MSISDN
&service_id=SERVICEID
&client_trans_id=UNIQID
&async=ASYNC
&desc=DESC
&key=KEY		 HTTP GET
LOGIN your login
MSISDN subscriber number
SERVICEID service id (will be supplied)
UNIQID uniqid for every request to prevent multiple submission the same request
ASYNC 0 - synchronous, 1 - asynchronous
DESC charing purpose (max 2000)
KEY md5 of (md5 of your password) + LOGIN + MSISDN + SERVICEID + UNIQID + ASYNC

Answer will be in xml format as followed:

<?xml version="1.0" encoding="utf-8"?>
<result>
<code>CODE</code>
<operator_id>OPERATOR</operator_id>
<orig_code>ORICODE</orig_code>
<transaction_id>TRANSACTION</transaction_id>
</result>
Parameters
CODE current operation response code
OPERATOR id of the mobile operator (look below)
ORICODE if duplicate request response code of original operation
TRASACTION transaction id
