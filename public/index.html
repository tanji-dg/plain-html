Project Overview<

 Work with The Bonneville Shop of Denver, Colorado to create a web application that assists in inventory control, management, and prediction.

Project Fundamentals

- Design and test a stand-alone application (Windows 10/ Linux) - Inventory prediction and seasonality analysis. - Individual SKU supplier pricing, with historical price deviations - Create and follow project specifications using software engineering guidelines, ethics, and practices.
