Result code for Bulk platform

Value (Dec-Hex)

Description

0 - 0

Message is successfully accepted by BulkSMS platform.

1101 - 44D

Campaign is not approved.

1102 - 44E

User credentials are not valid (login/password).

1103 - 44F

User SMS limit reached.

1104 - 450

Invalid request arguments.

1105 - 451

SMPP connection is down with SMSC.

1106 - 452

Allowed sending rate reached.

1107 - 453

Originator is not from listed

1108 - 454

Msisdn is not in Active users list

1109 - 455

Msisdn is in Blacklist (Global BlackList)

1110 - 456

Msisdn is in user defined Blacklist

1111 - 457

Msisdn is refused from bulk sms (User Defined in BlackList)

1113 - 459

Internal BulkSMS platform error.

1114 - 45A

Out of Working Hours.
