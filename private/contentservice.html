CONTENT SERVICE API

Operation URL Method Params details Response Additional information
subscribe by text message http://apps.lsim.az/contentservice/restful/subscribers/subscribe?
msisdn=xxx-xxx-xx-xx&
text=xxxx&
user=xxxx&
password=xxxx&
format=xxxx 		HTTP GET
username (required) your username on lsim
password (required) your password on lsim
msisdn (required) subscriber number (country code + operator code + number)
text (required) message sent by subscriber
format (optional) if you want to get result as boolean send format=bool then
you will get response as 0 or 1.If there is some success operation such as successfully subscribed,unsubscribed so you will get 1 otherwise 0
you will get json object: {"errorCode":null,"errorMessage":null,"obj":"29","successMessage":null}

successMessage Success message of operation
obj integer that represents transaction id
errorMessage Error message of operation if some error occured
errorCode If some error occured then it will show error code

check subscriber's subscribtion history http://apps.lsim.az/contentservice/restful/subscribers/subscribtionhistory?
msisdn=xxx-xxx-xx-xx&
subserviceid=xx&
subscribtionstatus=xx&
user=xxxx&
password=xxxx 		HTTP GET
msisdn (required) subscriber number (country code + operator code + number)
subserviceid (optional) select subscriber's subscribtion history of only this subservice id
subscribtionstatus (optional, default ONLY_ACTIVE) ONLY_ACTIVE - select only active subscribtions
ONLY_DEACTIVE - select only deactive subscribtions
ALL - select all subscribtions
username (required) your username on lsim
password (required) your password on lsim
you will get json object: {"errorCode":null,"errorMessage":null,"obj":[],"successMessage":null}

successMessage Success message of operation
obj list describes subscribtions.Include subscriber information and subservice information susbcribed
errorMessage Error message of operation if some error occured
errorCode If some error occured then it will show error code
