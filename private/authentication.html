sending authentication code http://apps.lsim.az/lsimrest/authentication/sendcode?
username=test&
password=test&
msisdn=xxx-xxx-xx-xx&
sendername=xxxx 		HTTP GET
username your username on lsim
password your password on lsim
msisdn subscriber number (country code + operator code + number)
sendername from which number code will be sent
you will get json object: {"errorCode":null,"errorMessage":null,"obj":"29","successMessage":null}

successMessage Success message of operation
obj integer that represents transaction id
errorMessage Error message of operation if some error occured
errorCode If some error occured then it will show error code

password your password on lsim

successMessage Success message of operation
obj boolean that represents code entered right
errorMessage Error message of operation if some error occured
errorCode If some error occured then it will show error code
